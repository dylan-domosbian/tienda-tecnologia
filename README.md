# Tienda de Tecnología

Este es un proyecto estictamente de **maquetación y diseño UI/UX**. No posee funcionalidades de backend ni base de datos. El objetivo principal fue demostrar el dominio de herramientas de estilizado avanzado y diseño responsivo.

## Tecnologías utilizadas
* **HTML5** (Estructura semántica)
* **SASS** (Arquitectura de estilos y variables)
* **Bootstrap 5** (Sistema de grillas y componentes)
* **Bootrstap Icons** (Iconografía)
